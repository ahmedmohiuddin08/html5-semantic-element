Margin

margin is a css property and used to create space around the elements, outside the border of the element.

You can specify margin for each side using four individual properties.

These includes:
  1. margin-top
  2. margin-right
  3. margin-bottom
  4. margin-left

    margin property can be set in one single line.

    For exemple:
  • element { margin: 10px 15px 20px 25px; }

    The first value represents margin-top. And the rest of the value represents margin-right, margin-bottom and margin-left respectively. Clockwise.

  • element { margin: 10px 15px 20px; }

    In this case, the first value represents margin-top, the second value represents margin-right and margin-left and the third value represents margin-bottom.

  • element { margin: 10px 15px; }

    And here the first value of the margin property represents margin-top and margin-bottom and the second value represents margin-left and margin-right.

  • element { margin: 25px; }

    This means that the value of 25px of margin will equally added around the element.

  • margin:auto will horizontally center the element within its container.
